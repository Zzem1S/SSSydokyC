# Sudoku Solver
Sudoku Solver - алгоритм, предназначеный для решения судоку с различным количеством пустых полей. В данном проекте реализован алгоритм, решающий судоку стандартным методом перебора пустых полей, и подбором необходимых значений путем исключения уже занятых цифр в строках, столбцах и клетках.

------

## Структура проекта
В первую очередь проект состоит из двух файлов main.cpp. Один из них отвечает за запуск самой программы для решения судоку. Второй предназначен для запуска тестов, проверяющих корректность работы кода.
Созданы следующие файлы для выполнения алгоритма:
1. input.cpp - Содержит функцию для выполнения ввода данных.
2. output.cpp - Содержит функцию для вывода решения.
3. solve.cpp - Содержит функции, благодаря которым производится заполнение пустых ячеек судоку.
4. utils.cpp - Содержит прочие функции, позволяющие преобразовывать данные в нужный формат (векторы) и т.д.
К каждому файлу .cpp идет файл .h, содержащий имена функций, к которым идет обращение из других файлов.
Так же содержится отдельная папка tst с файлами, покрывающими тестами функции выполнения алгоритма.

## Функционал
После запуска программы необходимо вписать в терминал необходимое для решения поле судоку (пустые клетки обозначать как "0") и нажать "enter". 
В результате при корректном вводе поля судоку будет выведено уже решенное поле со всеми заполенными ячейками.
